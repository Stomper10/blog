Put the Korean file in `src/content/notes/ko/` and the English file
with the same filename in `src/content/notes/en/` to form a pair.
The two files must share the same `translationKey` for the language
toggle to link them.

This file is a sample that demonstrates the structure — feel free to
delete it when you write your first real note.
